# Fichas de Verificação — WME Engenharia

PWA (app de página única) para Fichas de Verificação de Serviço (FVS) e de Material (FVM).

- **Site (GitHub Pages):** https://pelegrinianderson41.github.io/qualidade-obras/
- **Dados (texto):** JSONBin · **Fotos:** Supabase Storage
- **Publicar:** rodar `Publicar-GitHub.ps1` (copia a fonte, incrementa o Service Worker e dá push).

> Arquivos servidos: `index.html`, `sw.js`, `manifest.json`, `icon.svg`.
